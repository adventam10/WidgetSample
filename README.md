# WidgetSample
Android Studio 3.6.3 の Layout Editor の Palette に表示される UI コンポーネントのサンプル集。

Google と Layoutsは除外。

下記は実装予定

* AppBarLayout
* BottomAppBar
* NavigationView
* BottomNavigationView
* Toolbar
* TabLayout
* TabItem
* NavHostFragment
